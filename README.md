# [project name : Edu-Platform Landing Page]

Tani waa bog casri ah oo loogu talagalay barashada xirfadaha internetka, kaas oo aan ku dhisay **React**, **TypeScript**, iyo **Tailwind CSS**.

## ✨ Features (Waxa uu qabanayo)
- **Responsive Design:** Wuxuu u muuqdaa mid xirfad leh Mobile iyo Desktop-ba.
- **Modern UI:** Waxaan isticmaalay Tailwind CSS si aan u helo muuqaal qurxoon.
- **Interactive Menu:** Waxaa ku jira menu-ga mobile-ka (hamburger menu).

## 🛠️ Tech Stack (Agabka)
- React + Vite
- TypeScript
- Tailwind CSS
- Lucide-react (Icons)
